다이나믹셀 X 시리즈는 로보티즈 액츄에이터의 새로운 제품군으로써, 고성능의 네트워크 액츄에이터 모듈입니다. 신뢰성 있는 다관절 로봇을 제작하는데 널리 사용되고 있으며 모듈형태이기 때문에 확장성이 매우 좋습니다.
